# goshazaytsev-is-31
# Органайзер дел
Приложение для отслеживания дел и напоминаний

## Причины разработки
Удобство отслеживания дел и установка нужных напоминаний с добавлением заметок

## Функционал
- Добавление/удаление дел и напоминаний
- Календарь
- Заметки
